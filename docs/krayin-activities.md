# Krayin Activities

Source: [Krayin User Documentation - Activities](https://docs.krayincrm.com/2.x/activity/activities.html)

## Overview
- Activities capture scheduled interactions (calls, meetings, lunches) tied to leads.
- Default activity types: Lunch, Call, Meeting.
- Each activity stores a title, description, participants, schedule window, and location.

## Create an activity
1. Go to `Leads >> Open Lead >> Activity`.
2. Fill the activity form:
   - **Activity**: Choose Call, Lunch, or Meeting.
   - **Title**: Short subject for the activity.
   - **Description**: Details for the interaction.
   - **Participants**: People involved in the activity.
   - **Schedule**: From/To date and time.
   - **Location**: Where the activity happens.
3. Click **Save Activity** to add it to the lead.

## Views and management
- Activities appear under **All** plus type-specific sections (Lunch, Call, Meeting).
- Calendar View shows activities on their scheduled date/time for quick discovery.
- Actions let users edit or delete existing activities.
- Filters include Title, Created By, Lead, Schedule From, Schedule To, and Created At.

## Use in this project
- Mirror the Lunch/Call/Meeting defaults when scheduling lead activities.
- Require schedule windows and surface participants/location fields on create/edit.
- Provide list filtering across title, owner, lead, schedule range, and created date, plus a calendar view for date-based discovery.
