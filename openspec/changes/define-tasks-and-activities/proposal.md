# Proposal: define-tasks-and-activities

## Change ID
- `define-tasks-and-activities`

## Summary
- Capture Tasks lifecycle requirements including creation/assignment, due dates, priorities, statuses, completion tracking, reminders, categories, and recurrence.
- Define Notes capabilities across CRM entities with rich text, attachments, privacy controls, categorization, and history.
- Specify Activity feed/timeline expectations for comprehensive history, chronological display, filtering/search, activity typing, and automated logging.

## Capabilities
- `task-lifecycle-and-organization`: Create, assign, schedule, prioritize, categorize, remind, complete, and recur tasks with status tracking.
- `note-capture-and-privacy`: Attach rich text notes with files, categories, visibility, and historical context to any CRM entity.
- `activity-feed-and-discovery`: Provide chronological, filterable, searchable activity feeds covering calls, emails, meetings, notes, and automated logging.

## Notes
- OpenSpec CLI validation is not available in this environment; specs are authored manually.
