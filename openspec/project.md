# Project Overview

## Name
CRM Platform (Phase 1)

## Description
- Laravel 12-based CRM with Filament, Livewire, and modern frontend tooling.
- Covers Accounts, Relaticle modules, jobs, and integrations (Horizon, Mailcoach, Flowforge).
